## 🚀 快速启动

### 方式一（最简单）：直接打开
1. 找到 `Dashboard` 文件夹
2. 双击打开 `index.html` 文件
3. 应用在浏览器中启动

### 方式二（推荐）：使用本地服务器

#### Windows 用户
```powershell
# 进入 Dashboard 文件夹
cd "...\Dashboard"

# 使用 Python 启动服务器
python -m http.server 8000

# 或使用 Node.js
npx http-server
```

然后在浏览器访问：`http://localhost:8000`

## 💡 使用示例

### 修改某个月的销售数据
1. 点击 **数据管理** 标签页
2. 在 **月份销售** 标签中找到对应月份
3. 修改 "销售金额" 列
4. 点击 **保存数据**
5. 返回仪表板查看更新

### 添加新的员工
1. 点击 **数据管理**
2. 切换到 **员工销售** 标签页
3. 点击 **添加行** 按钮
4. 输入员工名称和销售数据
5. 点击 **保存数据**

### 删除某个地区
1. 点击 **数据管理**
2. 切换到 **地区销售** 标签页
3. 找到要删除的地区行
4. 点击该行的 **删除** 按钮
5. 点击 **保存数据**

## 🔧 常见操作

### 保存数据
- 修改完所有数据后点击 **保存数据**
- 数据将被保存到浏览器 LocalStorage
- 刷新页面数据不会丢失

### 重置数据
- 点击 **重置为默认** 恢复初始数据
- ⚠️ 此操作无法撤销

### 查看图表
- 柱状图支持悬停显示数值
- 地图可缩放和拖动
- 排名表自动排序
