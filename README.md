# 销售数据仪表板

一个功能完整的 Web 应用仪表板，用于展示和管理销售数据。

## 功能特性

### 1. 📊 仪表板视图

#### ① 近12个月销售金额柱状图
- 展示过去12个月的销售趋势
- 使用 Chart.js 库实现交互式柱状图
- 悬停显示具体销售金额
- 自动格式化为万元单位展示

#### ② 中国地图销售分布
- 展示全国各地区本月销售分布
- 使用 ECharts 绘制可视化地图
- 支持地区销售金额的直观展示
- 气泡大小代表销售金额多少

#### ③ 员工销售排名表
- 展示本月员工销售金额排名
- 支持同比增长百分比计算
- 排名前三使用金银铜牌标记
- 绿色/红色显示增长/下降状态

#### ④ 关键指标卡片
- **总销售额**: 本月总销售金额
- **平均单笔**: 平均销售金额
- **销售订单数**: 本月订单总数
- **销售人数**: 销售团队规模

---

### 2. 📝 数据管理页面

三个标签页用于编辑不同类型的销售数据：

#### 月份销售
- 编辑过去12个月的销售数据
- 支持添加新月份和删除记录
- 直接修改销售金额

#### 地区销售
- 管理各地区本月销售数据
- 支持添加新地区和删除地区
- 实时更新地图展示

#### 员工销售
- 管理销售人员的销售数据
- 编辑当月和上月销售额
- 系统自动计算同比增长

## 使用说明

### 方式一：直接打开 HTML 文件
```bash
# Windows 用户
# 在 Explorer 中找到 index.html，双击打开即可
# 或使用：
start index.html
```

### 方式二：使用本地服务器（推荐）
```bash
# 使用 Python 3
python -m http.server 8000

# 或使用 Python 2
python -m SimpleHTTPServer 8000

# 或使用 Node.js (如果安装了 http-server)
npx http-server
```

然后在浏览器中访问：
```
http://localhost:8000
```

> **注意**: 某些浏览器中使用本地服务器可以更好地加载数据和第三方库

## 功能操作指南

### 查看仪表板
1. 点击导航栏的 **仪表板** 按钮
2. 查看四个模块的数据可视化

### 编辑数据
1. 点击导航栏的 **数据管理** 按钮
2. 选择要编辑的标签页（月份销售/地区销售/员工销售）
3. 在表格中修改数据或使用 **添加行** 按钮增加新记录
4. 点击 **删除** 按钮删除不需要的行
5. 修改完成后点击 **保存数据** 按钮

### 重置数据
- 点击 **重置为默认** 按钮可以恢复初始数据
- 此操作无法撤销，请谨慎使用

---

## 数据存储机制

### 本地存储
- 编辑的数据保存在浏览器的 LocalStorage 中
- 刷新页面数据不会丢失
- 清空浏览器缓存会丢失数据

### 初始数据
- `data.json` 文件包含应用启动时的默认数据
- 点击 **重置为默认** 可以恢复

### 导出数据
在浏览器开发者工具中执行：
```javascript
console.log(JSON.stringify(dashboardData, null, 2))
```

## 常见问题

### Q: 数据为什么没有保存?
A: 检查浏览器是否启用了 LocalStorage，某些隐私模式可能禁用存储功能。

### Q: 地图不显示怎么办?
A: 确保网络连接正常，某些地区可能需要翻墙才能加载 CDN 资源。

### Q: 如何备份数据?
A: 在数据管理页面，打开浏览器开发者工具，在 Console 中运行上述导出命令。
